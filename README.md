# V2bX
A V2board node server powered by sing-box / hysteria cores.

一个基于 sing-box / hysteria2 内核的 V2board 节点服务端，支持 V2ay、Trojan、Shadowsocks 等协议。

Find the source code here: [baimi-bm/testv2bx](https://github.com/baimi-bm/testv2bx)

如对脚本不放心，可使用此沙箱先测一遍再使用：https://killercoda.com/playgrounds/scenario/ubuntu

# 详细使用教程

[教程](https://v2bx.v-50.me/)

# 一键安装

```
wget -N https://raw.githubusercontent.com/baimi-bm/V2bX-script/main/install.sh && bash install.sh
```
